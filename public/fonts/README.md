# Self-hosted fonts

These WOFF2 files are the Latin and Latin Extended variable subsets returned by the existing Google Fonts request on 2026-07-15:

- Geist `100..900` (`v5`)
- Geist Mono `100..900` (`v6`)
- Lora normal and italic `400..700` (`v37`)

They remain licensed under the bundled SIL Open Font License files. When updating a font, replace its files, update the versioned filenames and `@font-face` declarations, then increment each affected applet's offline catalogue version.
