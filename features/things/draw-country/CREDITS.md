# Country outline data

The generated country outlines are derived from Natural Earth’s public-domain
1:50m Admin 0 Countries dataset (version 5.1.2). The checked-in data contains the
195 commonly recognised sovereign states, simplified and normalised for this
drawing game. Regenerate it with `node scripts/generate-country-outlines.mjs`.

For France, the Netherlands, Norway, Portugal, Spain, Chile, and Ecuador,
distant overseas territories are omitted so the familiar home outline remains
large enough to draw. Nearby islands remain part of each outline.

Source: https://www.naturalearthdata.com/downloads/50m-cultural-vectors/
